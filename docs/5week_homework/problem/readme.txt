숙제 제출 방법
첨부 파일을 받아주세요.
압축을 풀어 VS Code에서 폴더 열기로 폴더를 열어주세요.
아래 실습 요구 사항과 파일을 확인하여 비워져 있는 코드를 채워주세요.
폴더의 이름을 [강의 제목_이름]으로 변경한 후, 폴더를 압축하여 제출해주세요.
예시 : 코딩이 처음이어도 쉽게 배우는 웹개발 A to Z_김르탄_동적 데이터.zip

제출 시 주의 사항
반드시 코드를 모두 작성하고, 제출 전에 코드 정렬을 진행해 주세요.
코드 정렬
Windows : Shift + Alt + F
macOS : Shift(⇧) + Option(⌥) + F
클래스와 아이디명은 스네이크 케이스를 따라주세요.
스네이크 케이스란 모든 단어를 소문자로 표기하고, 단어를 언더바(_) 로 연결하는 방법을 의미해요.
예시 : team_sparta , i_love_you
